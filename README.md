# 🚀 Automation Testing Learning Repository

This repository contains my journey and hands-on practice in **Automation Testing**. It includes examples, concepts, and real-time scenarios implemented using industry-standard tools and frameworks.

## 📚 What You'll Find Here

- ✅ Selenium WebDriver with Java
- ✅ Test case automation for web applications
- ✅ Handling web elements (forms, buttons, dropdowns, alerts, frames)
- ✅ Waits (Implicit & Explicit)
- ✅ XPath & CSS Selectors
- ✅ TestNG basics and annotations
- ✅ Data-driven testing concepts
- ✅ Exception handling in automation
- ✅ Screenshot capturing and reporting
- ✅ Real-time practice examples

## 🛠️ Tools & Technologies

- Java
- Selenium WebDriver
- TestNG
- Maven (optional)
- ChromeDriver
- IDE: Eclipse / IntelliJ / STS
