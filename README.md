# 🚀 Automation Testing Framework - Selenium + TestNG

## 📌 Project Overview

This project is a **Hybrid Test Automation Framework** built using **Selenium WebDriver, TestNG, and Java** to automate web application testing.

It follows industry best practices like **Page Object Model (POM)** and supports scalable, maintainable, and reusable test automation.

---

## 🎯 Key Features

* ✅ Page Object Model (POM) design pattern
* ✅ Data-driven testing using Excel (Apache POI)
* ✅ Cross-browser testing support (Chrome, Edge)
* ✅ Screenshot capture on test failure
* ✅ Logging using Log4j
* ✅ TestNG framework integration
* ✅ Parallel test execution
* ✅ Reusable utility methods

---

## 🛠️ Tech Stack

* **Language:** Java
* **Automation Tool:** Selenium WebDriver
* **Testing Framework:** TestNG
* **Build Tool:** Maven
* **Logging:** Log4j
* **Data Handling:** Apache POI
* **IDE:** Eclipse / IntelliJ

## 📂 Project Structure

```
Automation-Testing/
│── src/test/java
│   ├── testcases        # Test classes
│   ├── pages            # Page Object classes
│   ├── utilities        # Utility classes
│   └── base             # Base setup
│
│── src/test/resources
│   ├── testdata         # Excel data files
│   └── config           # Configuration files
│
│── testng.xml           # TestNG configuration
│── pom.xml              # Maven dependencies
```

## ⚙️ Setup Instructions

### 1️⃣ Clone the repository

```bash
git clone https://github.com/Jaswanth778012/Automation-Testing.git
```

### 2️⃣ Import project

* Open in IntelliJ / Eclipse as a Maven project

### 3️⃣ Install dependencies

```bash
mvn clean install
```

### 4️⃣ Run tests

```bash
mvn test
```
